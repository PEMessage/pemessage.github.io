九.TFT-LCD

FPGA Note
  1. LCD接口描述
    1. RGB888: 24根数据线
    2. DE: Date Enable 数据使能线
    3. VSYNC: 垂直同步信号线
    4. HYSNC: 水平同步信号线
    5. PCLK: 像素时钟信号线
  2. 配置
    1. 扫描方式配置
      🖼️通过焊接选择扫描方式
      1. FPGA不专门控制对应的数据线来控制扫描方式
      2. 而是将对应LCD控制线LCD_LR/LCD_LR接入1 或 0即可
      3. LCD_LR=1: Left→Right(横屏看)
      4. LCD_UD=0: UP→Down(横屏看)
    2. LCD_ID
      🖼️LCD_ID与复用上下拉关系
      1. 没有专门的存储器存储
      2. 通过复用数据线R7 G7 B7,添加上下拉电阻来分辨LCD_ID
  3. LCD扫描流程
    🖼️一帧扫描流程
    1. HSYNC: 是水平同步信号,当产生此信号的话就表示开始显示新的一行了
    2. VSYNC: 是垂直同步信号,当产生此信号的话就表示开始显示新的一帧图像了
    3. CRT历史原因
      1. 显示完一行→HSYNC 信号此时电子枪就会关闭，然后迅速的移动到屏幕的左边，当HSYNC 信号结束以后就可以显示新的一行数据了，电子枪就会重新打开。
