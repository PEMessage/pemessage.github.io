一.Bash
参考网址
bash keyboard shortcuts - Linux - SS64.com
Ctrl + a Go to the beginning of the line (Home) Ctrl + e Go to the End of the line (End) Ctrl + p Previous command (Up arrow) Ctrl + n Next command (Down arrow) Alt + b Back (left) one word Alt + f Forward (right) one word Ctrl + f Forward one character Ctrl + b Backward one character Ctrl + xx Toggle between the start of line and current cursor position Ctrl + L Clear the Screen, similar to the clear command Alt + Del Delete the Word before the cursor.

位置移动
Forward Backward
C-(Forward) C-Backward 左右移动 : (字母)
Alt-Forward Alt-Backward 光标驻留点将
C-Pervious C-Next 上下移动 等价方向键上下
C-Ahead C-End 移动到行首行尾 等价Home End
删除操作
基本删除Delete
C-Delete(字母D) 删除光标指向字母
Alt-Delete(字母D) 删除单词 (向右)
删除 源光标 和 最近向右驻留点 之间内容
原光标指向字母被删除 向右驻留点指向不被删除( [ ... ) 形式)
基本反向(左边)删除
C-H 删除光标之前的字母 等价Backspace
C-Word 删除单词 (向左)
光标指向之前 到 最近的左移动驻留点
原光标指向字母被不被删除 向左驻留点指向被删除( [ ... ) 形式)
删除句子
C-U 向左
C-K 向右
均为( [ ... ) 形式)
撤销: C-_
二.作用区
三层作用区: 字母 单词 行
两个光标方向: 光标前 光标后(含光标)
移动驻留点:
对于向 左 移动,光标总是驻留在单词第一个字母
对于向 右 移动,光标总是驻留在单词最后一个字母之后
[ ... ) 形式 : 即删除操作时候,对于右边光标指向不删除,而左边光标指向被删除的形式
