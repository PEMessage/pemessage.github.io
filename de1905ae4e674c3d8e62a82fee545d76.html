四.一般文本处理
一.文本显示
cat tac
b -n 列出Row number前者不包括Blank row
不可打印显示
E 显示句末$断航符号
T 显示Tab按键以 ^I 形式
v 显示其他不可打印
tac 同 cat 但是从最后一行开始
nl
该命令为cat -b/-n 细化
b 行号显示规则
b a 等价cat -n 对blank row显示行号
b t 等价cat -b
n 行号显示方式
n ln 显示于左
n rn 显示于右边
n rz 显示于右边,并根据字段数补零如 001 002 00001 00002
w 指定字段数 默认为6
more
n N 向后/前查询
:f 打印行号文件名
od
非文本文件的显示
t [type]
a -c 默认字符编码输出/ASCII编码输出
d[size] 10进制输出 [size]为一整数占用字节数目下同
f[size] 浮点
o[size] (Octal)8进制
x[size] (Hexadecimal)(Hex)16进制
比较输出
如od -t oCc 同时输出ASCII和八进制比较
