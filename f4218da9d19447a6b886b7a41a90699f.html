FPGA Note
  1. 时钟IP核MMCM(混合模式时钟管理) PLL(锁相环) 简介
    1. 🖼️IP核结构
    2. 参考文档: 7系列FPGA时钟资源(UG472)
    3. 包括 全局时钟(Global) 区域时钟(Regional)
      1. 全局时钟: 专用全局网络 有专用时钟缓存和驱动结构→到其他逻辑资源(BRAM)延迟低
      2. 区域时钟: 只能驱动区域内部的时钟资源和IO口
  2. 时钟结构图

    🖼️单个时钟区域放大图
    🖼️CMT结构框图
    🗒️名词解析
    1. BUFG: Buffer Global 全局时钟缓冲
    2. BUFH: Buffer H(???) 水平时钟缓冲
    3. BUFIO: Buffer IO IO时钟缓冲
    4. BUFR: Buffer Region 区域时钟缓冲
    5. BUFMR: Buffer Multi-Region多区域时钟缓冲
    6. MMCM: Multi Mode Clock Controller 混合模式时钟管理器
    7. PLL: 锁相环
    8. CC: 时钟专用引脚(接外部输入)
      1. SRCC仅本区域
      2. MRCC相邻区域
      3. P N区别: 可以接受差分信号,单端时钟仅能连到P端
    1. BUF作用域
      1. BUFG可以到达时钟域(Clock Region)任何角落,而BUFH只能到HROW左右相邻的时钟区域
      2. BUFIO: 只能作用于IO区域,而不能在内部逻辑区域使用
      3. BUFR: 只能作用于本区域,相当于BUFH阉割
      4. BUFMR: 不仅可以作用于本区域,也可通过CMT Backbone作用相邻区域
      5. CMT: MCMM PLL
  3. Clock Management Tiles(CMT)
    1. CMT简介
      1. 一个CMT 包括一个MMCM和一个PLL
      2. 7020 4个CMT \ 7010 2个CMT
      3. Clock Frequency Synthesis时钟合成
      4. Desked 倾斜校正
      5. Jitter Filtering 过滤抖动
    2. MCMM/PPLL内部结构
      🖼️MCMM结构框图
      🖼️PLL结构框图
      1. MCMM实际为PLL功能升级: 分数倍频率综合 反向时钟输出 动态时钟相位调整
      2. D(可编程计数分频)→PFD(相位 频率检测器)→CP(电荷泵)→LF(环路滤波器)→VCO(压控振荡器)→Phase-Tab相位抽头
      3. Lock信号: 锁相信号

    1. IP Category→Clock→Clock Wizard
    2. Clock Wizard配置

        2. 配置Feature 选择时钟源
