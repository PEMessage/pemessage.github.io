五.RAM IP核

FPGA Note
  1. RAM资源简介
    🖼️Zynq7000系列资源介绍
    1. 参考文献: Zynq存储资源指导手册(DS190)
    2. 7020-140片-4.9M | 7010-60片-2.1Mb
    3. RAM端口设置
      1. 为双端口,但是需要定义读优先还是写优先
      2. 也可配置单端口: 只有一个端口
      3. 伪双端口: 一个写一个读
      4. 真双端口: 两个端口都可以读写
    4. PL每片RAMBlock为36Kbits,每片RAM独立端口,读写独立,时钟独立
    5. 端口位宽配置
      1. 可配置32K*1—16K*2—8K*4—4K*8(9)—2K*16(18)—1K*32(36) 最大位宽为512*72
      2. 不是所有的位宽都可以充分利用36Kbit
      3. 也可配置16K*1 512*36只使用16(18)Kbit
      4. 双端口位宽都可变
      5. 两片RAM邻近时候也可组合使用64K*1
  2. RAM_IP核(Block Memory Generator)单端使用
    1. RAM Wizard配置
      1. Basic
        1. Interface Type接口选择: Native AXI4
        2. Memory Type类型选择: RAM/ROM 双端口 单端口,此处为单端口RAM示例
        3. ECC纠错功能: 单端口无法使用,需要伪双端口(Simple Dual Port)或者其他
        4. Write Enable: Byte Write Enable以字节(8 或 9)为单位更新,不需要
        5. 生成倾向: 最小资源-低功耗-固定源语
        6. 使用空间大小
      2. Port A Option设置
        1. 读写位宽 读写深度
        2. 访问同一地址时候,要求写优先还是读优先
        3. 是否需要使能引脚
        4. 输出寄存器缓冲:- 优点: 时序上严谨 缺点: 增加延迟
      3. Other Option/Summary: 如果是ROM需要加载Coe(Coefficient) File
