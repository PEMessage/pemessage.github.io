三.基本文件操作
一.一般处理
cd
pwd -P
P选项为显示当前真实路径而非Link路径
mkdir
m 手动配置权限(manual)不使用默认(umask)
例如mkdir -m 711 [dir]
p 递归创建目录
rmdir
p 同mkdir,递归创建
ls
显示内容
a -A 列出所有文件
A选项不包括. .. 目录
d 仅目录本身
R 列出包括子文件的内容
排序相关
f 不排序
r 反向排序
S 大小排序
t 时间排序
显示属性相关
F 给予附加结构
可执行
/ Dir
= Socket File
| FIFO File
i 显示inode
l 显示多项数据包括
n 显示UID GID
-time={atime,ctime,mtime} 时间选择
cp mv
注意事项
是否需要
链接相关
l 硬链接
s 软链接
d (direct) 复制.lk 文件而非其指向文件
更新相关
u 按时间更新
f 强制更新
i (interface)交互更新
p (property)属性
a 等价-pdr 常用于完全备份
rm
i 同上
f 忽略警告信息 同上
r 递归 同上
basename dirname
basename /dir1/dir2/file 取得文件名 此处为[file]
basename /dir1/dir2/file 取得目录名 此处为[/dir1/dir2]
touch
修改项
a 修改acess time(atime) 读取文件后刷新
m 修改modification time(mtime) 更改文件后刷新
c 若文件不存在则创建
修改值
r [file1] [file2] copy [file1] 时间到 [file2]
t 修改为YYMMhhmm 如0810011212
d "time"修改为想要的日期,格式如下
“2 days ago”
