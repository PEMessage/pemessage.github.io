一.Zynq简介

FPGA Note

一.综合简介

  1. SoC简介
    1. 跟版上系统对比,把各种模块集成到单芯片上SoC解决方案成本更低
    2. ASIC-SoC:
    3. SOPC(System on Programmable Chip): 纯FPGA搭建的系统,软核microblaze && NIOS II
    4. APSoC(All Programmable SoC): 存在处理器软件软件可编程 FPGA硬件可编程

二.Programmable Logic

  1. 可编程逻辑器件(Programmable Logic Device PLD):
    1. 允许用户电路修改内部连接
    2. PLD内部电路结构可编程,可擦除重写
  2. 常见PLD
    1. CPLD(Complex PLD): 基于乘积项与或逻辑阵列
    2. 现场可编程门阵列FPGA: 基于查找表(Look Up Table LUT)的 可配置逻辑块(Configurable Logic Block CLB) 阵列
  3. CLB单元构成(Xilinx 7系列)
    🖼️CLB简易图示
    1. 两个Slice
      1. 每个Slice含有4个LUT和一些触发器
  4. FPGA架构
    🖼️FPGA架构图示
    1. 可编程输入逻辑: IO-BlockDDR双倍速率 串并转换
    2. 可编程逻辑单元
    3. 硬核: XADC
    4. 底层嵌入单元: DSP 锁相环PLL
    5. 嵌入式RAM(Block RAM)
    6. 布线资源

三.Process S

🖼️图示
  1. 应用处理单元APU
    1. 双核ARM Cortex-A9
    2. FPU浮点单元: 浮点运算加速
    3. 单指令多数据NEON引擎: 比如视频中对每个像素的并行处理 比如FFT
    4. MMU存储管理单元: 物理地址到虚拟地址映射
    5. 指令Cache 数据Cache : 为L1 处理器独立
    6. Snoop Control Unit 窥探控制单元: 维护不同核的Cache一致性 控制L2-Cache OCM片上存储器
    7. DMA
    8. 中断控制器GIC
  2. APU之外
    1. 中央互联器 OCM互联 PL互联
    2. 外设接口→MIO映射复用(区别于PL的IOBlock)→Pin(54个)
    3. 动态存储器接口: DDR2/3 静态存储器接口: SRAM Flash
    4. 时钟生成 复位模块
  3. PS到PL接口
    1. EMIO: 可连接到IOBlock 最简单的将IOBlock用作PS的引脚
    2. Conifg AES/SHA配置接口
    3. XADC: 12位
    4. DMA
    5. IRQ: 连到GIC中断控制
  4. AXI (Advanced Extensible Interface)
    🖼️AXI图示
    🖼️AMBA组成
    1. AMBA简介
      1. 属于AMBA: Advanced Microcontroller Bus Architecture
    2. ACP:
      1. PL均位主机
      2. 直接连接SCU 实现PL到L2的访问,实现PL到PS低延迟访问
    3. General-Purpose Ports
      1. Master: M_AXL_GP[1:0] PS作为主机
      2. Slave: S_AXL_GP[1:0] PS作为从机
    4. High Performance Ports:
      1. 相对于GP而言,PL均位主机
      2. 存在Buffer FIFO,以实现高带宽
